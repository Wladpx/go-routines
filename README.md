# go-routines

Pequeno projeto desenvolvido para estudar concorrência em Go utilizando goroutines e `sync.WaitGroup`.

O programa realiza múltiplas requisições HTTP simultaneamente para verificar o status de diferentes serviços e medir o tempo total de execução.

---

## 🚀 Conceitos praticados

- Goroutines
- Concorrência em Go
- `sync.WaitGroup`
- Requisições HTTP
- Tratamento de erros
- Medição de tempo de execução
- Status HTTP

---

## 🛠 Tecnologias

- Go (Golang)
- `net/http`
- `sync`
- `time`
- `fmt`

## ▶️ Como executar

Clone o repositório:

```bash
git clone <https://github.com/Wladpx/go-routines.git>
```

Entre na pasta do projeto:

```bash
cd go-routines
```

Execute o programa:
```bash
go run main.go
```
